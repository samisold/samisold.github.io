Hexo系列：用hexo在github和coding上同时搭建个人博客
Hexo
文章 2018-12-27

捣鼓一天多时间搭建博客，现在开始整理思路，把这个过程记录下来，个人主页预览

概述

用hexo在github和coding上同时搭建，对国外和国内IP进行自动分流，提高访问速度
大概分为以下几步

  1. 搭建环境准备
  2. 安装、配置hexo
  3. 注册、设置github&coding
  4. 设置ssh
  5. 博客上传
  6. 域名注册与绑定
  7. 发布文章

建站步骤

搭建环境准备

  • 安装Node.js（官方下载）
  • 安装Git（官方下载）安装设置默认即可，过程略
    git --version查看版本号
    node -v查看版本号

安装、配置hexo

安装Hexo

  • 在d盘新建文件夹git（作为博客目录）
  • win+R打开运行窗口，输入cmd，打开cmd
    cd\
    d:
    cd git
  • 安装hexo
    npm install hexo-cli -g
  • 查看版本号
    hexo -v

设置Hexo

  • 接上一步
    hexo init blog(此处有一个坑，init后面必须设置目录否则会出错提示no empty，“blog”是我放博客的目录，可以自定义)
    cd blog(记住切换到博客目录再操作)
    npm install
  • 安装hexo主题（美化）
    hexo主题站
    可以选择自己喜欢的主题进去找到克隆地址然后克隆到本地
    cd blog(切换到博客目录)
    git clone https://github.com/huweihuang/hexo-theme-huweihuang.git ./hexo-huweihuang
  • 本地预览
    hexo g(生成静态页面)
    hexo s(打开本地服务器)
    在浏览器中打开http://localhost:4000/，可以看到安装好的博客页面

    注册设置github&coding

  • 注册过程略
  • 创建代码库


**github库名必须为"用户名.github.io"**  
**coding库名必须为"用户名.coding.me"**  
附：coding需手动打开pages服务
<img title="图片标题" alt="图片说明" class="class1 class2" src="http://pk535c0ah.bkt.clouddn.com/static//images/2018-12-20-09-47-42.png">

SSH授权(github&coding双公私秘钥)

  • 本地生成秘钥
    ssh-keygen -t rsa -C "your_email@example.com" #github/coding注册邮箱

    成功后显示

    Generating public/private rsa key pair.
    Enter file in which to save the key (/Users/xxx/.ssh/id_rsa):
    #第一次输入/Users/xxx/.ssh/id_rsa_coding
    #第二次输入/Users/xxx/.ssh/id_rsa_github

  • 进入.ssh文件夹，创建config文件，并配置

    github

    Host github.com
    HostName github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_github

    coding.net

    Host git.coding.net
    User youremail.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_coding

上传博客

  • .md博客文件存储目录为”博客根目录/source/_post”
    笔者是直接在此目录中用VS Code管理和编写博客(也可以用hexo new命令新建博客)。

  • 　打开blog下配置文档_config.yml，对deploy:字段进行如下修改

    deploy:

    类型

    type: git

    仓库

    repo:

      github: git@github.com:samisold/samisold.github.io.git
  coding: git@git.coding.net:samisold/samisold.coding.me.git
  #gitee: git@gitee.com:samisold/samisold.git

    分支

    branch: master

注意：type、repository、branch冒号的后面都有一个空格

  • 右键点击blog目录，选择Git Bash

  • npm install hexo-deployer-git --save
    hexo g
    hexo d
    上传完毕
