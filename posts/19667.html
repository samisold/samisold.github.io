Hexo系列：Hexo主题matery的自定义修改
Hexo
文章 2018-12-27

Matery官方库

  • Matery
  • 帮助文档

把主页默认的英文菜单改成中文

进入主题目录\themes\hexo-theme-matery，打开_config.yml文件，修改为

menu:
  首页:
    url: /
    icon: fa-home
  分类:
    url: /categories
    icon: fa-bookmark
  标签:
    url: /tags
    icon: fa-tags
  归档:
    url: /archives
    icon: fa-archive
  关于:
    url: /about
    icon: fa-user-circle-o
  友情链接:
    url: /friends
    icon: fa-address-book

修改首页页脚版权声明

进入目录\themes\hexo-theme-matery\layout\_partial，打开footer.ejs文件，修改为

<footer class="page-footer bg-color">
    <div class="container row center-align">
        <div class="col s12 m8 l8 copy-right">
            &copy;2011-2018 <a href="http://www.syteng.com" target="_blank">常青滕’s Blog</a> 保留所有权利
        </div>
        <div class="col s12 m4 l4 social-link"><%- partial('_partial/social-link') %></div>
    </div>
</footer>

修改logo、favicon和微信二维码等

进入目录themes\hexo-theme-matery\source，themes\hexo-theme-matery\source\medias，themes\hexo-theme-matery\source\medias\reward，修改相应文件即可

新建Page

新建分类 categories 页

categories页是用来展示所有分类的页面，如果在你的博客source目录下还没有categories/index.md文件，那么你就需要新建一个，命令如下：

hexo new page "categories"

---
title: categories
date: 2018-09-30 17:25:30
type: "categories"
layout: "categories"
---

新建标签 tags 页

tags页是用来展示所有标签的页面，如果在你的博客source目录下还没有tags/index.md文件，那么你就需要新建一个，命令如下：

hexo new page "tags"
编辑你刚刚新建的页面文件/source/tags/index.md，至少需要以下内容：

---
title: tags
date: 2018-09-30 18:23:38
type: "tags"
layout: "tags"
---

新建关于我 about 页

about页是用来展示关于我和我的博客信息的页面，如果在你的博客source目录下还没有about/index.md文件，那么你就需要新建一个，命令如下：

hexo new page "about"
编辑你刚刚新建的页面文件/source/about/index.md，至少需要以下内容：

---
title: about
date: 2018-09-30 17:25:30
type: "about"
layout: "about"
---

新建友情连接 friends 页（可选的）

friends页是用来展示友情连接信息的页面，如果在你的博客source目录下还没有friends/index.md文件，那么你就需要新建一个，命令如下：

hexo new page "friends"
编辑你刚刚新建的页面文件/source/friends/index.md，至少需要以下内容：

---
title: friends
date: 2018-12-12 21:25:30
type: "friends"
layout: "friends"
---

修改社交链接

在主题文件的/layout/_partial/social-link.ejs文件中，你可以修改或添加你需要的社交链接地址

其中，社交图标在Font Awesome中搜索找到。以下是常用社交图标标识:

Twitter: fa-twitter
Google-plus: fa-google-plus
Linkedin: fa-linkedin
Tumblr: fa-tumblr
Medium: fa-medium
Slack: fa-slack
新浪微博: fa-weibo
微信: fa-wechat
QQ: fa-qq

自定义主页文字，解除与subtitle和description的绑定

进入博客根目录，在_config.yml文件中添加字段

# 自定义的matery_Theme首页文字  
matery_subtitle: 常青滕’s Blog
matery_description: 念念不忘，必有回响
  • 打开目录themes\hexo-theme-matery\layout\_partial\bg-cover.ejs文件
  • 将config.subtitle和config.subtitle字段替换为config.matery_subtitle和config.matery_subtitle即可

自定义主页文章卡片中的摘要内容

  • 修改目录\themes\hexo-theme-matery\layout\目录下的index.ejs、archive.ejs、category.ejs文件
  • 将<div class="summary"><%- strip_html(post.content).substring(0, 70) %>修改为<div class="summary"><%- strip_html(post.excerpt).substring(0, 70) %>
  • 在编写文章的时候，在文章中插入标签<!-- more -->，标签<!-- more -->以上的内容为文章摘要，摘要中的前70个字符现在在主页文章卡片中

美化文章中超链接，添加变色效果和椭圆边框

打开/source/css/my.css，如果主题没有自定义css功能，可以在主题css中添加内容
/ 给超链接增加特效 /
.dt-p {
background-color: white;
border: 1px solid #333;
border-radius: 45px;
width: auto;
height: auto;
padding: 9px 45px 9px 45px;
text-align: center;
display: inline; /后面无换行，宽度随内容变化/
}

    .dt-p:hover {
        background-color: black;
    }

    #dt-a {
        text-decoration:none !important; /*去掉超链接下划线*/

    }

    #dt-span {
        color: white;
        mix-blend-mode: difference;  /*颜色反相*/
    }

博文中需要加效果的链接代码如下(class、ID名称与css中一致)

<div>
        <a href="/posts/41119.html" id=dt-a><p class=dt-p><span id=dt-span>Windows软件推荐</span></p></a><br>
        </div>
